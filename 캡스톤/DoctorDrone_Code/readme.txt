#Doctor Drone
드론에 표준행동에 의한 자율주행을 통해 상대적 공간좌표를 구해
드론 프로펠러의 이상 유무를 탐지

##Installation
플랫폼 메인의 라즈베리파이에 

DD_FFT_MIC.ino파일을 연결된 아두이노에 설치합니다.
DD_Drone_reflecter.ino파일은 드론에 들어갈 아두이노에 설치합니다.
DD_processing.pde파일은 라즈베리파이에서 실시간으로 마이크에 들어오는 입력값을
FFT로 분해된 값으로 볼 수 있게 해줍니다.(Processing 소프트웨어가 필요)
Buzzer_tester.c 는 라즈베리파이에 연결된 스피커 소리를 테스트 하는 파일입니다.

Mobius폴더안의 action#.py파일들은 플랫폼에서 얻어진 센싱값들을 모비우스 서버에 데이터를 전송하는 코드입니다.


###사용 방법
드론과 플랫폼을 블루투스로 동기화 시킨 뒤, 플랫폼의 버튼을 누르면,
자동으로 공간좌표를 측정하며, 측정된 센싱값은 자동으로 Mobius서버에
데이터가 보내집니다.
보내진 데이터는 사용자 디바이스에서 DoctorDrone 소프트웨어를 통해
분석한 결과들을 볼 수 있습니다.
